Resources page of rk0cc's open source project

This website provides necessary documentations (APIs, guides) and online demo of public GitHub repositories based on adapted package manager. There is a table that it lists availability of resources depending on package manager system of specific programming language:

Programming language Package manager Resources type
Document Demo
Dart Pub ✓ ✓
Java Maven ✓ ✗
Gradle ✗ ✗

Dart

Pub is package manager for Dart language, which has official hosting website pub.dev by default. It integrates API documentations, which generated after publishing packages. Majority of released packages can be found in this page.

Some Flutter packages also offers online demostration on website to familiar usages before implement into project. There are fews online demo pages were hosted already:

collection_change_notifier

Notify changes when collection object altered.

github_colour_flutter

Colour picker based on programming language colours from GitHub

Java

Java has two different package manager, which namely Maven and Gradle. Maven is the one and only package manager adapted in rk0cc.xyz that every Java package documentations are based on Maven. Most Javadoc of published xyz.rk0cc packages can be found in self-hosted. However, it only contains stable releases packagaes due to complexity of uploading Javadoc. As a result, there is a chance of finding unstable release in javadoc.io.
